# Simple Todo App

This is a lightweight Todo Manager that allows users to create, display, and remove tasks with optional due dates. Todos are fully interactive â€” they include completion toggling, validation, and dynamic UI rendering.

## Functionality

The Todo application provides the following features:

Add new tasks through a modal popup

Each task includes:

Task title

Optional due date (formatted using locale rules)

Completion checkbox

Delete button

Validation ensures user input is valid before submission

Existing tasks load from predefined initial data

Tasks dynamically render using reusable templates and OOP structure

## Technology

This app is built using core web technologies and best practices:

Frontend:

HTML5 semantic structure

CSS3 for styling

JavaScript (ES6+) modular files

Object-Oriented Programming for Todo and FormValidator

DOM manipulation

UUID for unique todo IDs

Architecture & Practices:

Custom reusable Form Validation Class

Separation of concerns:

Components in /components

Page logic in /pages

Static configuration in /utils

Template cloning for consistent, scalable rendering

Popup modal management
